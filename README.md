# FCUP-VC

## Installation

[Python 3](https://www.python.org/) must be installed and the following libraries are required:
* [NumPy](https://numpy.org/)
* [OpenCv](https://opencv.org/get-started/)
* [Kociemba](https://pypi.org/project/kociemba/)


## How to run

    ```
    python3 main.py
    ```
